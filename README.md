## Requirement

-   PHP version 8.2.0
-   Laravel version 10.41.0

## Instalation Project

1.   Rename file ".env.example" menjadi ".env"
2.   Buka terminal, ketik "npm i"
3.   "npm add -D sass"
4.   "npm run dev"
5.   Buka terminal baru, jalankan "php artisan serve"
